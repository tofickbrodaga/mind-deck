# MindDeck Frontend

Фронтенд приложение для MindDeck - системы изучения с карточками на основе алгоритма FSRS.

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- React Query
- Zustand
- Tailwind CSS
- Axios

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

## Структура проекта

```
src/
  ├── components/      # Переиспользуемые компоненты
  ├── pages/          # Страницы приложения
  ├── services/       # API сервисы
  ├── store/          # Управление состоянием (Zustand)
  ├── types/          # TypeScript типы
  └── App.tsx         # Главный компонент с роутингом
```

## Функциональность

- ✅ Аутентификация (регистрация, вход)
- ✅ Управление колодами (создание, редактирование, удаление)
- ✅ Управление карточками (создание, редактирование, удаление)
- ✅ Импорт карточек (Word, Excel, изображения с OCR)
- ✅ Режимы изучения:
  - Флэшкарты
  - Множественный выбор
  - Письмо
  - Подбор (сопоставление)

## Переменные окружения

Создайте файл `.env` в корне проекта `frontend/`:

```
VITE_API_URL=http://localhost:8000/api/v1
```

По умолчанию используется `http://localhost:8000/api/v1`, если переменная не задана.

## Быстрый старт

1. Установите зависимости:
```bash
cd frontend
npm install
```

2. Убедитесь, что бэкенд запущен на порту 8000

3. Запустите фронтенд:
```bash
npm run dev
```

4. Откройте браузер по адресу http://localhost:3000

## Основные страницы

- `/login` - Вход в систему
- `/register` - Регистрация
- `/` - Список колод
- `/decks/:id` - Детали колоды с карточками
- `/decks/:id/study` - Режимы изучения
- `/decks/:id/edit` - Редактирование колоды
- `/cards/:id/edit` - Редактирование карточки
